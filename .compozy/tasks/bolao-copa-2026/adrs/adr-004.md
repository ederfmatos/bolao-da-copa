# ADR-004: React SPA com Vite, deploy na Vercel

## Status
Accepted

## Context
Precisamos de um frontend mobile-first, rápido de desenvolver e fácil de deployar. Avaliamos Next.js, Remix, e React puro com Vite.

## Decision
Usar React SPA (Single Page Application) com Vite como build tool e deploy na Vercel:
- **Vite**: build rápido, HMR instantâneo, config mínima
- **React Router**: roteamento client-side
- **Supabase JS Client**: comunicação com backend
- **Vercel**: deploy automático via GitHub, edge network global

## Consequences
- **Positivo**:
  - Setup mínimo, foco em features
  - Vite é significativamente mais rápido que Webpack/CRA
  - Vercel free tier é generoso e inclui preview deployments
  - SPA é suficiente para este caso (sem necessidade de SSR/SSG)
- **Negativo**:
  - SPA carrega todo o bundle inicial (menos crítico para app focado)
  - SEO não é relevante para este caso (app autenticado)
- **Risco**: Bundle size pode crescer; mitigado por code splitting se necessário
