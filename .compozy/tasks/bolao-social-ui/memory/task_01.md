## Objective Snapshot

Install and configure Tailwind CSS v3 with PostCSS, Autoprefixer, dark mode class strategy, and custom color palette.

## Important Decisions

- Used Tailwind CSS v3 (not v4) because v4 drops `tailwind.config.js` and `darkMode: 'class'` support. The installed v4 was replaced with v3.
- Used CommonJS (`module.exports`) in tailwind.config.js because `package.json` has `"type": "commonjs"`.

## Learnings

- Tailwind CSS v4 does not support `npx tailwindcss init -p` or `tailwind.config.js` — the v3 CLI is required for the config-based setup this project needs.
