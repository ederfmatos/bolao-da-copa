# Install Tailwind CSS and configure

## Overview

Install and configure Tailwind CSS with PostCSS and Autoprefixer. Set up the base CSS file with Tailwind directives and configure dark mode support.

<critical>
- Read TechSpec "Tailwind CSS Configuration" section before starting.
- Focus on configuration only — no component migration in this task.
- Ensure dark mode is set to 'class' strategy.
- Tests are required.
</critical>

<requirements>
1. MUST install `tailwindcss`, `postcss`, and `autoprefixer` as dev dependencies.
2. MUST run `npx tailwindcss init -p` to generate config files.
3. MUST configure `tailwind.config.js` with `darkMode: 'class'`.
4. MUST add custom colors as defined in TechSpec (primary, accent, dark palette).
5. MUST create `src/index.css` with Tailwind directives.
6. MUST import `index.css` in `src/main.jsx`.
7. SHOULD verify build completes without errors.
</requirements>

## Subtasks

- [ ] Install tailwindcss, postcss, autoprefixer.
- [ ] Initialize Tailwind config with `-p` flag.
- [ ] Configure tailwind.config.js (content paths, darkMode, custom colors).
- [ ] Create src/index.css with @tailwind directives.
- [ ] Import index.css in main.jsx.
- [ ] Run `npm run build` to verify setup.
